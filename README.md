# SerpientesyEscaleras

The snake and ladder game with UI and customizable multiple players.

This java platform project allows us to play the famous snakes and ladders game with a friend and customize both players with previously loaded icons.
And you will can choose the name that each one wants.

And you can see the progrees of the game on the UI, the UI shows both players on the board and their progress.
